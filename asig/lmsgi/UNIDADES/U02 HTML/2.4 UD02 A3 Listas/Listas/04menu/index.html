Elemento Menú

El elemento menu representa una barra de menú que consiste en una lista de comandos representada por una lista no ordenada de ítems (elementos li) . Este elemento es simplemente una alternativa semántica al elemento ul que se utiliza para representar una lista no ordenada de comandos. Por tanto, los elementos <menu> y <ul> representan una lista desordenada de elementos. La diferencia clave es que <ul> contiene principalmente elementos para mostrar, mientras que <menu> estaba destinado a elementos interactivos.
