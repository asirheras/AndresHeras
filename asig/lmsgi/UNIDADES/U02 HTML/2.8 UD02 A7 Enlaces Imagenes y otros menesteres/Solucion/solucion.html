Etiqueta ancla (anchor) <a>

En general, sólo se debe utilizar un ancla (anchor) para navegación utilizando una URL adecuada. No deben usarse para la creación de pseudo botones

Cuando se clicea sobre un <a> las consecuencias varían de acuerdo al navegador y el sistema operativo.
